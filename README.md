# Future_Interns_CyberSecurity

This repository contains the tasks completed during my Cyber Security Internship at Future Interns.

## Tasks Included

Task 1 – Vulnerability Assessment Report for a Live Website  
- Performed vulnerability scanning and analysis using tools like Nmap and OWASP ZAP.
- Identified security issues and classified risks (Low / Medium / High).
- Provided remediation steps and security recommendations.

Task 2 – Phishing Email Detection & Awareness System  
- Analyzed phishing email samples and identified common phishing indicators.
- Classified emails as Safe, Suspicious, or Phishing.
- Created awareness guidelines to help users detect and prevent phishing attacks.

Task 3 – API Security Risk Analysis  
- Analyzed API endpoints to identify potential security risks.
- Checked authentication, authorization, and data exposure issues.
- Provided recommendations to improve API security.

## Tools Used
- Nmap
- OWASP ZAP
- Wireshark
- Browser Developer Tools
- Postman / API Testing Tools
- Kali Linux

# FUTURE_CS_03 – API Security Risk Analysis

## Overview
This project contains a professional API Security Risk Analysis conducted as part of the Future Interns Cyber Security Internship (2026).

The assessment focuses on identifying authentication, authorization, data exposure, and rate-limiting risks in a public demo API.

## Tools Used
- Postman
- Browser Developer Tools
- Manual Risk Analysis

## API Tested
JSONPlaceholder  
https://jsonplaceholder.typicode.com

## Key Findings
- Unauthenticated endpoint access
- Excessive data exposure
- Missing object-level authorization
- Lack of rate limiting controls
